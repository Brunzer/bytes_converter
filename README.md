# BytesConverter

Gem converting Kilobytes, Megabytes and Gigabytes into bytes.

## Installation

Add this line to your application's Gemfile:

    gem 'bytes_converter'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bytes_converter
